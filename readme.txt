## Beta Priyoko's Tonjoo Test 1 & 2
- Test 1 & 2 Saya kerjakan di 1 apps dengan berbeda route karena alasan efisiensi, efektivitas waktu, dan resource, untuk kemudahan development dan pengecekan.

### Informasi
Node Version : `v20.16.0`
NPM Version: `10.8.3`
Next.js Version: `14.2.8`

### How to:
  1. Buka `CMD/Terminal` dan masuk ke direktori folder `/test-1-dan-2`
  2. Jalankan `npm install` untuk menginstall seluruh dependensi yang di butuhkan aplikasi.
  3. Untuk Development mode jalankan `npm run dev`
  4. Untuk Production mode jalankan `npm run build` tunggu sampai selesai kemudian jalankan `npm start`
  5. Akan muncul url `http://localhost:3000` salin urlnya kemudian buka di browser.
